# 狼人杀群聊版（多 AI + 用户）

## 你提到的关键需求已实现
- 主界面为群聊布局，右侧玩家列表配置。
- 10 名玩家每个都可单独配置 OpenAI 兼容 API（Base URL / API Key / Model）。
- 每个玩家都有“测试通信”按钮。
- 没有 API 配置不允许开始游戏。
- 取消本地随机兜底策略：AI 决策必须走对应玩家 API。

## 流程与规则
服务端状态机按以下环节推进：
1. 狼人刀人（四狼独立投票，平票随机）
2. 预言家验人
3. 女巫使用技能（不可自救）
4. 猎人操作（被刀/公投可开枪，吃毒不可）
5. 白痴操作（被公投翻牌不死，失去投票权）
6. 警长竞选（首日）
7. 发言
8. 投票（警长 1.5 票权）
9. 遗言

此外支持狼人白天自爆（进入黑夜，警上自爆则竞选顺延）。

## 运行
```bash
npm install
npm start
```
打开：`http://localhost:3000`
